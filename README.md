# Jest Tests

Jest Tests project was created for study purposes.

## Installation

Make sure you have Node.js/NPM installed (LTS version is recommended). [Download](https://nodejs.org)

```bash
npm install
```

## Usage

Run the tests: 

```bash
npm run test
```
